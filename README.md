# client

## Project setup
You need to install dependices separately for the server and client side. To install, execute the following code snippets:
### Installation
#### Windows (Powershell)
```
cd server; npm install; cd ..\client; npm install; cd ..
```

#### Linux
```
cd server && npm install && cd ../client && npm install && cd ..
```

### Start Server App
```
npm run server
```

### Start Client App
```
npm run client
```
